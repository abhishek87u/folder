Definition/Description list

HTML
This is the markup language for web pages. It defines the meaning and structure of web content.
CSS
This is the language used to style the HTML document. It describes how HTML elements should be displayed.
JavaScript
This is the programming language of the web. It is used to program the behavior of web pages.
Python
This is used on the server-side to create web applications.
